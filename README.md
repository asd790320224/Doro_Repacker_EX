1. 安装pycharm等IDE（或用CMD的python），直接在当前目录（Doro_Repacker_EX）下新建项目
2. 执行pip install -r requirements.txt安装代码的必备环境，同时安装NET.9.0（https://dotnet.microsoft.com/zh-cn/download/dotnet/9.0），好像也要安装8.0，根据报错再一步一步看吧。
3. 按顺序执行Z0 → Z1 → Z2 → Z3 (→ Z4），如果有问题先看调试信息去查错！
4. Repacked里就有最终mod了
